📊 DEPLOYMENT ARCHITECTURE

LOCAL (Development):
  Your Machine
  ├─ Docker Container (API on :5000)
  ├─ Docker Container (Frontend on :80)
  └─ SQL Server (Local on :1433)

AZURE (Production):
  Internet
  ├─ Container Instance (Frontend, Public IP)
  │  └─ http://ems-frontend-xxxxx.azurecontainers.io
  └─ Container Instance (API, Public IP)
     └─ http://ems-api-xxxxx.azurecontainers.io:5000
        └─ Azure SQL Database (or Local SQL)

🔐 SECURITY NOTES

Before going to production:
  • Change JWT secret from default (Jwt__Secret)
  • Use strong SQL database password
  • Enable HTTPS with Application Gateway
  • Set CORS to specific domains only
  • Enable Azure's built-in security features

See: FINAL_DEPLOYMENT_SUMMARY.md → Security Configuration
